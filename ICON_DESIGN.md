# PhoneBuddy Icon Design

## Icon Concept

The PhoneBuddy icon represents **remote control of audio profiles** between two devices.

### Design Elements:
- **Two connected phones** (representing the pair)
- **Remote control symbol** (play/pause/volume icon) in the center
- **Audio waves** or **sound bars** to represent audio control
- **Connection line** between devices

### Color Scheme:
- **Primary**: Blue (#5E5CE6) - matches the "Ringing" profile color
- **Secondary**: Orange (#FF8A65) - matches the "Vibrate" profile color  
- **Accent**: Red (#EF5350) - matches the "Silent" profile color
- **Background**: Gradient from blue to purple

### Icon Style:
- Modern, minimalist design
- Rounded corners (Android Material Design)
- Clear visibility at small sizes
- Distinctive and memorable

## Implementation

The icon should be created as:
- **Foreground**: Two stylized phones with a control symbol
- **Background**: Gradient circle with brand colors
- **Size**: 512x512px source, then scaled to Android requirements

## Android Icon Sizes Required:
- mipmap-mdpi: 48x48px
- mipmap-hdpi: 72x72px
- mipmap-xhdpi: 96x96px
- mipmap-xxhdpi: 144x144px
- mipmap-xxxhdpi: 192x192px

## Design Tools:
- Figma, Adobe Illustrator, or Inkscape for vector design
- Export as PNG at required sizes
- Ensure transparency/alpha channel for adaptive icons
